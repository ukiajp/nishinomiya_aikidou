# 設計 - 西宮合氣道同好会 公式サイト

## カラーパレット

| 役割 | Tailwind クラス | HEX目安 |
|---|---|---|
| ベース（白） | `bg-white` | #FFFFFF |
| メイン（深紺） | `bg-slate-800` `text-slate-800` | #1E293B |
| アクセント（金） | `bg-amber-500` `text-amber-500` | #F59E0B |
| サブ文字 | `text-slate-500` | #64748B |
| 薄背景 | `bg-slate-50` | #F8FAFC |

## フォント方針

- 日本語：システムフォント（`font-sans`）
- 見出し：`font-bold`、やや大きめ（`text-3xl`〜`text-5xl`）
- 本文：`text-base`（16px）以上を厳守

## ページ構成とコンテンツ設計

### / トップページ

```
[Hero]
- 背景：紺のグラデーション or 道着の写真（placeholder）
- H1：「西宮で、はじめての合氣道」
- サブコピー：「年齢・体力は問いません。稽古を通じて、心と体を整える時間を。」
- CTA1：「無料体験を申し込む」（amber-500ボタン）
- CTA2：「稽古日程を見る」（アウトラインボタン）

[概要セクション]
- 3つの特徴をカードで紹介
  1. 何歳からでも始められる
  2. 競技ではなく「稽古」
  3. 西宮市内・通いやすい場所

[稽古風景]
- placeholder画像 × 2〜3枚
- 「稽古の様子」キャプション

[活動紹介へのリンク]
- 曜日・時間の概要 + 「詳しく見る」ボタン

[CTA バナー]
- 「まず見学してみませんか？体験稽古は無料です」
- 「体験を申し込む」ボタン
```

### /about 合氣道について

```
[ページヘッダー]
- 「合氣道とは」

[合氣道の特徴]
- 競い合わない武道：勝ち負けを目的としない

- 心身の調和：稽古を通じて精神的な落ち着きも育む

[他の武道との違い]
- 空手・柔道との比較表
  | | 合氣道 | 空手 | 柔道 |
  競技性 / 身体への負荷 / 年齢制限 / 西宮での普及

[初心者Q&A]
- 道着は最初から必要？
- 一人でも参加できる？
- 体力がなくても大丈夫？
- 何歳から始められる？

[CTA]
- 「稽古の様子を見てみる」→ /activity
```

## レイアウト方針

- モバイルファースト（`sm:` prefix で拡張）
- 最大幅：`max-w-4xl` / `max-w-6xl` + `mx-auto px-4`
- セクション間余白：`py-16` / `py-24`
- 文字サイズ：本文 `text-base`（16px）以上
- コントラスト：白背景に `text-slate-800`（WCAG AA 達成）
