# 要件定義 - 西宮合氣道同好会 公式サイト

## プロジェクト概要
西宮合氣道同好会の公式ホームページ。主目的は新規会員の獲得。

## ターゲット層

### メインターゲット
- **50代以上の男女**：健康維持・生涯スポーツを探している
- 運動から長く離れており、「自分にできるか不安」という気持ちを持つ
- 敷居の高い習い事に苦手意識がある

### サブターゲット
- 30〜40代で初めて武道に興味を持った人
- 子供の送迎をきっかけに自分も始めたい親

## ターゲットの不安と訴求ポイント

| 不安 | 訴求ポイント |
|------|------------|
| 体力がなくて続けられない | 体力・年齢不問、自分のペースでOK |
| 怪我しそう | 投げ合い中心ではなく、動きの稽古が主体 |
| ついていけるか不安 | 丁寧な指導、初心者クラスあり |
| 費用が心配 | 月謝が明確、体験稽古可能 |
| 雰囲気が怖い | 同好会なので気軽、先輩も親切 |

## 訴求コピー方針
- 「強くなる」より「整える」
- 「競技」より「稽古」「習慣」
- 「何歳からでも」「遅くない」を強調
- 先生・先輩の親切さ・指導の丁寧さを伝える

## CTA設計
1. **一次CTA**：「体験稽古に申し込む」（問い合わせページへ）
2. **二次CTA**：「稽古日程を確認する」（活動紹介ページへ）
3. すべてのページにCTAボタンを配置

## 必要ページ

| ページ | パス | 役割 | 主なCTA |
|--------|------|------|---------|
| トップ | `/` | 第一印象・概要・興味喚起 | 体験稽古申込 |
| 合氣道について | `/about` | 武道の特性・安全性・魅力 | 活動紹介へ |
| 活動紹介 | `/activity` | 稽古内容・スケジュール・場所 | 体験稽古申込 |
| 入会案内 | `/join` | 費用・流れ・FAQ | 体験稽古申込 |
| お問い合わせ | `/contact` | フォーム・アクセス情報 | - |

## コンテンツ上の注意
- 稽古場所の詳細は後で差し替え → プレースホルダー表示
- 稽古日は「週2回（日時は後で差し替え）」と記載
- 代表者名・電話番号は仮データ or 省略
- 画像はすべて placeholder.com 使用
